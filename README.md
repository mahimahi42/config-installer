config-installer
================

Installer for my personal configuration and setup.

Installs configurations for:
- tmux
- vim
- zsh

# Usage

## Installation

```shell
curl -sSL https://raw.githubusercontent.com/mahimahi42/config-installer/master/install.sh | sh
```

## Uninstallation

```shell
/home/<username>/.uninstall_configs
```
